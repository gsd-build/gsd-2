## Decisions Made
- **Static source-text for hook-using components:** PreviewPanel now uses `useState` internally; tests migrated to source inspection (Phase 12-01 pattern) since direct function calls fail in bun test environment without React context
- **No-cors fetch for port scanning:** mode: "no-cors" allows scanning running servers in browser context without CORS headers; AbortController at 800ms prevents scan hanging
- **Self-contained preview state:** PreviewPanelWithState owns all server state via usePreview() — AppShell only passes viewport and close/change callbacks
- **Direct localhost:PORT/ not proxy:** Using direct iframe src gives full interactivity; /api/preview/ proxy would introduce CORS restrictions
