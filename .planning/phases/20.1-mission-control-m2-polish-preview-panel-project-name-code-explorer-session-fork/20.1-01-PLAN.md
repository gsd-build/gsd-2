- Fix the derivation to:
```tsx
const projectName = activeProjectPath
```

- This ensures that even when `activeProjectPath` is set, if basename extraction fails for some reason, we fall back to `milestone_name`. More importantly, the `projectName` variable is now always derived so it is truthy when EITHER source has a name.

