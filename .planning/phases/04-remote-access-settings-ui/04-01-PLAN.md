  <action>
    Create `web/app/api/auth/password/route.ts` with:

    Top-level exports:
    ```
    export const runtime = "nodejs"
    export const dynamic = "force-dynamic"
    ```

    `export async function POST(request: Request): Promise<Response>`:

    1. Parse request body: `const { newPassword } = await request.json() as { newPassword: string }`
    2. Validate: if `!newPassword || newPassword.length < 4` return `Response.json({ error: "Password must be at least 4 characters" }, { status: 400 })`
    3. Hash password: `const hash = await hashPassword(newPassword)` (import from `../../../src/web/web-session-auth`)
    4. Store password hash: Use the same settings storage mechanism Phase 1 uses for `web.passwordHash`. Read the existing preferences/settings write pattern from `web/app/api/preferences/route.ts` to determine how to persist. Store the hash under key `web.passwordHash`.
    5. Rotate session secret: `const newSecret = await rotateSessionSecret()` — this invalidates all existing sessions (per D-07, AUTH-07)
    6. Create fresh session token: `const token = createSessionToken(newSecret)`
    7. Set cookie for current browser (per D-07 — current session stays logged in):
       ```
       const cookieStore = await cookies()
       cookieStore.set("gsd-session", token, {
         httpOnly: true,
         secure: true,
         sameSite: "strict",
         maxAge: 30 * 24 * 60 * 60,
         path: "/",
       })
       ```
    8. Return `Response.json({ ok: true })`

    Imports needed:
    - `import { cookies } from "next/headers"`
    - `import { hashPassword, rotateSessionSecret, createSessionToken } from "../../../../src/web/web-session-auth"`

    Per D-06: No current password verification required. Single-user tool — if the user is in settings, they are authenticated.

    `export async function GET(): Promise<Response>`:

    1. Import `getTailscaleStatus`, `isTailscaleInstalled` from `../../../../src/web/tailscale`
    2. Check if Tailscale is installed: `const installed = await isTailscaleInstalled()`
    3. If not installed: return `Response.json({ installed: false, connected: false, hostname: "", tailnetUrl: "", dnsName: "" })`
    4. Get status: `const info = await getTailscaleStatus()`
    5. If null (not connected): return `Response.json({ installed: true, connected: false, hostname: "", tailnetUrl: "", dnsName: "" })`
    6. Return `Response.json({ installed: true, connected: info.connected, hostname: info.hostname, tailnetUrl: info.tailnetUrl, dnsName: info.dnsName })`

    The response shape is:
    ```typescript
    interface TailscaleStatusResponse {
      installed: boolean
      connected: boolean
      hostname: string      // e.g., "mybox"
      tailnetUrl: string    // e.g., "https://mybox.tailnet.ts.net"
      dnsName: string       // e.g., "mybox.tailnet.ts.net" (trailing dot already stripped by tailscale.ts)
    }
    ```

    The `tailscale.ts` module from Phase 2 already strips the trailing dot from DNSName. This endpoint is a thin wrapper.
