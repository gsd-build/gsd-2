    Top-level exports:
    ```
    import { spawn } from "node:child_process"
    import { isTailscaleInstalled, getInstallCommand, getTailscaleStatus } from "../../../../src/web/tailscale"

    export const runtime = "nodejs"
    export const dynamic = "force-dynamic"
    ```

    Define the auth URL regex at module level:
    ```
    const AUTH_URL_PATTERN = /https:\/\/login\.tailscale\.com\/[^\s]+/
    ```

    Helper function `sendEvent(controller: ReadableStreamDefaultController, event: object)`:
    ```
    function sendEvent(controller: ReadableStreamDefaultController, event: object): void {
      controller.enqueue(new TextEncoder().encode(`data: ${JSON.stringify(event)}\n\n`))
    }
    ```

    Helper function `runStreamingCommand(cmd: string, args: string[], controller: ReadableStreamDefaultController): Promise<number>`:
    - Spawn the process with `spawn(cmd, args, { stdio: ["ignore", "pipe", "pipe"] })`
    - Pipe both stdout AND stderr (Pitfall 2: auth URL appears on stderr)
    - On each chunk from stdout: split by newline, for each non-empty line call `sendEvent(controller, { type: "output", line })`
    - On each chunk from stderr: split by newline, for each non-empty line:
      - Check if line matches AUTH_URL_PATTERN. If yes, extract the URL and call `sendEvent(controller, { type: "auth-url", url: match[0] })`
      - Always call `sendEvent(controller, { type: "output", line })` so stderr output is visible
    - Return a promise that resolves with the exit code on process `close` event
    - Handle process `error` event (e.g., command not found): sendEvent error and resolve with code 1

    `export async function POST(request: Request): Promise<Response>`:

    1. Parse body: `const { step } = await request.json() as { step: string }`
    2. Detect platform server-side (Pitfall 5): `const platform = process.platform` — never trust client-provided platform

    3. Create a ReadableStream and switch on `step`:

    **step === "detect":**
    - `const installed = await isTailscaleInstalled()`
    - `sendEvent(controller, { type: "detect", platform, installed })`
    - `sendEvent(controller, { type: "done", success: true })`
    - `controller.close()`

    **step === "install":**
    - `const [cmd, ...args] = getInstallCommand(platform)`
    - If platform is "darwin": cmd="brew", args=["install", "tailscale"]
    - If platform is "linux": cmd="sh", args=["-c", "curl -fsSL https://tailscale.com/install.sh | sh"]
    - Call `runStreamingCommand(cmd, args, controller)`
    - On exit code 0: `sendEvent(controller, { type: "done", success: true, message: "Tailscale installed successfully" })`
    - On non-zero exit: `sendEvent(controller, { type: "error", message: "Install failed with exit code ${code}. Check the output above." })`
    - `controller.close()`

    **step === "connect":**
    - `const code = await runStreamingCommand("tailscale", ["up"], controller)`
    - On exit code 0: `sendEvent(controller, { type: "done", success: true, message: "Tailscale connected" })`
    - On non-zero exit: `sendEvent(controller, { type: "error", message: "tailscale up failed with exit code ${code}" })`
    - `controller.close()`

    **step === "verify":**
    - `const status = await getTailscaleStatus()`
    - If status and status.connected:
      `sendEvent(controller, { type: "verify", connected: true, hostname: status.hostname, tailnetUrl: status.tailnetUrl })`
      `sendEvent(controller, { type: "done", success: true })`
    - Else:
      `sendEvent(controller, { type: "verify", connected: false })`
      `sendEvent(controller, { type: "error", message: "Tailscale is not connected. Try running the Connect step again." })`
    - `controller.close()`

    **default:**
    - `sendEvent(controller, { type: "error", message: "Unknown step: ${step}" })`
    - `controller.close()`

    4. Return `new Response(stream, { headers: { "Content-Type": "text/event-stream", "Cache-Control": "no-cache", "Connection": "keep-alive" } })`

    Important: Wrap the entire stream start() body in try/catch. On any unhandled error, sendEvent an error and close the controller.
