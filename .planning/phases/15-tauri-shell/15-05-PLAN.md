Prerequisites for `tauri dev` to work:
- Rust toolchain installed (`rustup`, `cargo`)
- Tauri CLI installed: `cargo install tauri-cli --version "^2.0"` or `npm install -g @tauri-apps/cli`
- The executor must have these installed. If not present, the verify command will fail — note this in the summary.

tauri.conf.json already has:
- `beforeDevCommand: "bun run --cwd packages/mission-control dev"` (from plan 15-01, Task 2)
- `devUrl: "http://localhost:4000"` (Tauri waits for this URL to be reachable before opening window)

The `tauri dev` command:
1. Runs beforeDevCommand (starts Bun server)
2. Waits for devUrl to respond
3. Compiles Rust in debug mode
4. Opens native window pointing at devUrl

The `tauri build` command:
1. Runs beforeBuildCommand (runs Bun build if configured)
2. Compiles Rust in release mode
3. Bundles app + produces installer
