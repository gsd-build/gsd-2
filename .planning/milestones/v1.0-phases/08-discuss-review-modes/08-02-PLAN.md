Create packages/mission-control/src/server/mode-interceptor.ts:

The function uses simple regex patterns for known fixed schema (per RESEARCH.md — no full XML parser needed). Use a buffer parameter to handle partial XML at chunk boundaries. Approach:

1. Prepend the buffer to incoming text to form scanText.
2. Search for known opening patterns: `<discuss_mode_start`, `<discuss_mode_end`, `<question `, `<decision `, `<review_mode_start>`.
3. For self-closing single-line tags (discuss_mode_start, discuss_mode_end, decision): use regex that matches the full tag. If matched, extract, emit event, remove from text.
4. For block tags (question, review_mode_start): find the closing tag `</question>` or `</review_mode_start>`. If found: extract the whole block, parse it, emit event. If closing tag not found: treat as partial — keep from the opening `<` onward in remainder.
5. Any text before the first mode tag (or all text if no mode tag found): goes into stripped.
6. After processing: strip = text with all complete mode blocks removed. remainder = any incomplete block waiting for next chunk.

For question parsing: regex to extract `id`, `area`, `type`, `number`, `total` attributes from the opening tag. Extract content between `<question ...>` and `</question>`. Within that, extract `<options>` block and parse each `<option value="...">Label</option>`. The question text is the content between the question opening tag and the first `<options>` (or end of block if no options).

For review_mode_start parsing: extract `<pillar name="..." score="...">` blocks and their `<finding>` children. Extract `<fix priority="..." pillar="...">text</fix>` blocks. Build ReviewResults object.

Export interface and function:
```typescript
import type { ModeEvent, ReviewResults, QuestionCardPayload } from "./chat-types";
export function parseStreamForModeEvents(
  text: string,
  buffer: string
): { events: ModeEvent[]; stripped: string; remainder: string }
```

Wire into pipeline.ts inside wireSessionEvents. Add a per-session buffer:
- Before the loop `session.processManager.onEvent(...)`, initialize `let modeBuffer = "";`
- Inside the handler, when `ev.type === "stream_event"` and `ev.event?.type === "content_block_delta"` and `ev.event?.delta?.type === "text_delta"` and `ev.event.delta.text`:
  1. Call `const { events, stripped, remainder } = parseStreamForModeEvents(ev.event.delta.text, modeBuffer);`
  2. Set `modeBuffer = remainder;`
  3. For each event in events: `wsServer.sendToClient(session.activeClient!, { ...modeEvent, sessionId: session.id });`
  4. If `stripped.trim() === "" && events.length > 0`: return early (suppress blank delta)
  5. If `stripped !== ev.event.delta.text`: mutate `ev.event.delta.text = stripped;`
  6. Continue to existing forwarding logic (no change to the forward path).

Import parseStreamForModeEvents at top of pipeline.ts.

export function QuestionCard({ question, onAnswer }: { question: QuestionCardPayload; onAnswer: (answer: string) => void }) {
  const [selectedAnswer, setSelectedAnswer] = useState<string | null>(null);
  const [freeTextValue, setFreeTextValue] = useState("");
  const [confirming, setConfirming] = useState(false);
  return (
    <QuestionCardView
      question={question}
      selectedAnswer={selectedAnswer}
      freeTextValue={freeTextValue}
      confirming={confirming}
      onSelectOption={setSelectedAnswer}
      onConfirm={() => { if (selectedAnswer) { setConfirming(true); onAnswer(selectedAnswer); }}}
      onFreeTextChange={setFreeTextValue}
      onFreeTextSubmit={() => { if (freeTextValue.trim()) onAnswer(freeTextValue.trim()); }}
    />
  );
}

interface DecisionLogDrawerProps {

export function DecisionLogDrawer({ decisions, visible }: DecisionLogDrawerProps) {
  if (!visible) return null;
  return (
    <div className="animate-in slide-in-from-right duration-200 w-48 border-l border-navy-600 bg-navy-900 flex flex-col flex-shrink-0">
      <div className="border-b border-navy-600 px-3 py-2">
        <span className="text-xs font-display uppercase tracking-wider text-slate-400">Decisions</span>
      </div>
      <div className="flex-1 overflow-y-auto">
        {decisions.length === 0 ? (
          <p className="text-xs text-slate-600 italic p-3">No decisions yet</p>
        ) : (
          decisions.map((d) => (
            <div key={d.questionId} className="border-b border-navy-800 px-3 py-2">
              <p className="text-xs text-slate-400 truncate">{d.area}</p>
              <p className="text-xs text-cyan-accent font-mono truncate">{d.answer}</p>
            </div>
          ))
        )}
      </div>
    </div>
  );
}
