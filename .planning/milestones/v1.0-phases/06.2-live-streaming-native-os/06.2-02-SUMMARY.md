# Phase 6.2 Plan 02: Sidebar Restructure + Single-Column View Summary

Sidebar-driven single-column view routing with ViewType discriminated union, replacing tab-based layout with ProjectTree navigation and extracted view components.

## What Was Built

### Task 1: ViewType + SingleColumnView + useSidebarNav + View Stubs

Created the navigation infrastructure:

- **view-types.ts**: `ViewType` discriminated union with 6 kinds (chat, activity, milestone, slice, history, verify)
- **useSidebarNav.ts**: Simple useState hook defaulting to `{ kind: "chat" }`
- **SingleColumnView.tsx**: Switch-based router rendering one view component per ViewType kind
- **ChatView.tsx**: Extracted "Chat & Task" tab content (TaskExecuting/TaskWaiting + ChatPanel)
- **MilestoneView.tsx**: Extracted milestone tab content (MilestoneHeader + PhaseList + CommittedHistory)
- **SliceView.tsx**: Extracted slice tab content (ContextBudgetChart + BoundaryMap + UatStatus)
- Activity, Verify, History render stub placeholders for Plan 03

### Task 2: Sidebar Rewrite + AppShell + ProjectTree

Rewired the layout:

- **ProjectTree.tsx**: Expandable tree with Chat, Activity, Milestones, Slices (with phase children), History, Verify items. Active item highlighted with cyan-accent border. Phase status badges.
- **Sidebar.tsx**: Rewritten with New Window button, Open Folder button (POST /api/dialog/open-folder), ProjectTree replacing NavItems+ProjectBrowser
- **AppShell.tsx**: Uses useSidebarNav, passes activeView to Sidebar and SingleColumnView. Handles project switch via POST /api/project/switch.
- Old components (TabLayout, NavItems, ProjectBrowser, FileTreeNode, useProjectBrowser) left as dead code -- not deleted to avoid breaking their own test files.

## Decisions Made

1. **Discriminated union routing over URL routing**: ViewType with `kind` field and switch statement is simpler and type-safe for a single-page app without deep linking needs.
2. **Dead code preservation**: Old components left in place (imports removed from Sidebar/AppShell) to avoid breaking unrelated tests. Cleanup deferred.
