    2. Create `fs-api.ts` with:
       - HIDDEN_DIRS constant: Set containing "node_modules", ".git", "$RECYCLE.BIN", "System Volume Information"

       - `handleFsRequest(req: Request, url: URL)` — route dispatcher for /api/fs/list (GET), /api/fs/detect-project (GET), /api/fs/mkdir (POST), returns Response or null. Wraps all readdir calls in try/catch, returns 400 for invalid paths, 403 for permission errors (EACCES/EPERM), 500 for other errors

       - RECENT_FILE constant: path.join(homedir(), ".gsd", "recent-projects.json")
       - `getRecentProjects()` — reads and parses file, returns [] on any error
       - `addRecentProject(project)` — deduplicates by path, prepends, trims to 20, creates dir, writes JSON
       - handleRecentProjectsRequest(req, url) — route dispatcher for GET/POST /api/projects/recent

    4. Write tests FIRST (TDD red phase) in tests/fs-api.test.ts and tests/recent-projects.test.ts using bun:test. For fs-api tests, create a temp directory structure with tmpdir(). For recent-projects tests, mock the RECENT_FILE path to a temp location.
