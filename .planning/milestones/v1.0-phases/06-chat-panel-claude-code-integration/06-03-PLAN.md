       - Import ClaudeProcessManager and routeMessage from server modules
       - Create a ClaudeProcessManager instance
       - Wire onChatMessage callback into createWsServer options:
         - Call routeMessage on the prompt
         - If type === "command": send system message back with command info
         - If type === "prompt": spawn Claude process, iterate stream events, forward each as chat_event via ws.send(), send chat_complete on finish
       - Pass repoRoot (resolve from planningDir) to spawn calls
       - Expose processManager on PipelineHandle for cleanup

    3. Update server.ts:
       - Ensure repoRoot is passed through to pipeline (already resolved, just verify)

    4. Create useChat.ts hook:
       - Manages chat state: messages array (ChatMessage[]), isProcessing boolean
       - useChat internally creates its own useReconnectingWebSocket connection to ws://localhost:4001 (same server, separate connection -- WS server supports multiple clients)
       - This avoids refactoring usePlanningState and keeps concerns separate
       - Sends chat messages via send(JSON.stringify({ type: "chat", prompt }))
       - Filters incoming messages by type:
         - "chat_event": accumulate streaming text into current assistant message, handle tool indicators from content_block_start (tool_use) and content_block_delta (text_delta)
         - "chat_complete": mark current message as done (streaming=false), set isProcessing=false, store sessionId
         - "chat_error": add system error message, set isProcessing=false
         - Ignores "full" and "diff" messages (those are for usePlanningState)
       - sendMessage(text): creates user ChatMessage, sets isProcessing=true, sends via WebSocket
       - Returns: { messages, sendMessage, isProcessing }

    5. Update AppShell.tsx:
       - Import useChat
       - Call useChat() -- it manages its own WebSocket connection
       - Pass chat props to TabLayout: chatMessages, onChatSend, isChatProcessing

       - Layout: ChatPanel takes flex-1, task info shows as a collapsible or compact area at top
